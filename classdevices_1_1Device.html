Detailed Description

Class that manages the MQTT and OPC UA communication for a device.

This class holds a reference to a bridge between MQTT and OPC UA, a dictionary of publisher/subscriber pairs for MQTT topics and OPC UA nodes, and a callback for OPC UA. The class manages subscriptions to OPC UA nodes and MQTT topics for the device, as well as handling incoming MQTT messages.

Handles MQTT messages.

This function is called whenever an MQTT message is received. It decodes the message, retrieves the relevant OPC UA nodes, typecasts the payload to the appropriate format for each node, and creates a task to write the value to each node.
