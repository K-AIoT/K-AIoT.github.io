Detailed Description

Class that acts as an OPC UA callback handler, managing any OPC UA events and data changes.

This class holds a reference to the device it is associated with and handles data changes and events coming from the OPC UA server. When a data change happens, it publishes the new value to all MQTT topics associated with the OPC UA node in the device.
