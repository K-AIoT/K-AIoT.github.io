Detailed Description

Arduino sketch to control a LCD display and send distance data over MQTT.

This sketch uses an ultrasonic sensor to measure distance. It then displays the distance on a 20x4 LCD and publishes it to an MQTT topic. Additionally, the LCD display shows incoming messages from a subscribed MQTT topic.

void createCustomCharacters ( )

This function creates custom characters to be used on the LCD display.

The LCD display used here has the capability to display custom characters. This function specifies the pixel pattern for each custom character and stores them in the LCD's memory. Each character is defined by an 8-byte array, with each byte representing a row of the character's 5x8 pixel matrix. A bit value of 1 in the array means that the corresponding pixel should be turned on.

Note
The custom characters are stored in the following indices:
  • 0: customChar0
  • 1: customChar1
  • 2: customChar2
  • 3: customChar3
  • 4: customChar4
  • 5: customChar5
  • 6: customChar6
  • 7: customChar7

int readRange ( )

Function to read range from the ultrasonic sensor.

This function sends a pulse from the ultrasonic sensor and measures the time it takes to return. The distance is then calculated using the speed of sound and returned as a rounded integer value.

Returns
Measured distance in cm.

< Sets the trigger pin to LOW.

< Delays for 2 microseconds to ensure a clean HIGH pulse.

< Sets the trigger pin to HIGH, sending out the ultrasound pulse.

< Waits for 10 microseconds while the pulse goes out.

< Sets the trigger pin to LOW again to end the pulse.

< Reads the echo pin, which will be HIGH for a duration proportional to the distance to the object.

< Calculate the distance by using the speed of sound and the round-trip time of the pulse.

< Prints the string "Distance (cm): " to the serial monitor.

< Prints the measured distance to the serial monitor.

< Rounds the float distance to an integer.

< Returns the measured distance.
