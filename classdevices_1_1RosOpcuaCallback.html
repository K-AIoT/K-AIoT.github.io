Asynchronous callback for data change events in the OPC UA server.

This method is called when a value change is detected in any OPC UA node. It then publishes the new value to each MQTT topic associated with the OPC UA node.

Parameters
nodeAn instance of the Node class from the asyncua module, which represents the OPC UA node that has a new value.
valThe new value in the OPC UA node.
dataAn instance of the DataChangeNotification class from the asyncua module.
