Detailed Description

Control a DC motor based on humidity data from a DHT22 sensor, with data published via MQTT.

Author
YourName

This Arduino sketch is used to control the speed of a DC motor depending on the humidity levels detected by a DHT22 sensor. The humidity data is also published to a MQTT broker.

The sketch is configured to work with an ESP8266, and it uses the PubSubClient library for MQTT communication and the DHT sensor library for interacting with the DHT22 sensor.

Handles the incoming messages on the subscribed MQTT topics.

This function is executed whenever a message is received on the subscribed MQTT topics. It prints the received message to the Serial monitor and checks if the topic is "dry/speed". If it is, it changes the speed of the motor based on the received message.

Parameters
topicThe topic on which the message is received.
messageThe received message.
lengthThe length of the received message.
Note
The function will convert the speed message from string to int and call the function changeSpeed to change the speed of the motor.

void changeSpeed ( int  speed)

Changes the speed of the motor based on the input value.

This function changes the speed of the motor based on an integer speed value passed in as a parameter. It also ensures the direction of the motor, in case it is switched later, and writes the speed to the ENA (Enable Pin A for Motor Driver) pin on the motor driver using PWM signal.

Parameters
speedThe desired speed value for the motor.
Note
The function will output a verification message to the serial monitor: "Jeg snurrer" (I'm spinning).

void loop ( )

The main function that is continuously looped after setup.

This function runs the MQTT client, checks if it is connected and tries to reconnect if not. It also sets up and checks measurement intervals, reads the humidity and temperature, and publishes these values to the MQTT broker. An analogRead is performed to measure voltage from a sensor which is also published.

Note
If the DHT sensor fails to read, the function will return early.
